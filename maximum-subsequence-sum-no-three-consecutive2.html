Understanding Subsequence Selection

A subsequence is a sequence that can be derived from another sequence by deleting some or no elements without changing the order of the remaining elements.

When selecting elements for a subsequence, there's an important rule: we cannot select three consecutive elements from the original array.

In the following slides, we'll explore examples of valid and invalid subsequence selections from this array.

Valid Subsequence Selection

This is a valid subsequence selection because:

  • We've selected elements at indices 0, 1, and 3
  • The selected elements are 1, 2, and 4
  • We did not select three consecutive elements

Notice that we selected elements at indices 0 and 1 (which are consecutive), but then we skipped index 2 and selected index 3. This breaks the pattern of consecutive selections, making it a valid subsequence.

Invalid Subsequence Selection

This is an invalid subsequence selection because:

  • We've selected elements at indices 0, 1, and 2
  • These indices are consecutive (0, 1, 2)
  • The selected elements are 1, 2, and 3

The rule states that we cannot select three consecutive elements from the original array. Since indices 0, 1, and 2 are consecutive, this selection violates the rule and is therefore invalid.
