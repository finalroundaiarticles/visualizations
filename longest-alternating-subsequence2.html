Dynamic Programming State Transitions

Understanding Inc and Dec Variables

This animation demonstrates how a dynamic programming approach tracks state transitions in an array. We're using two variables - inc and dec to track the lengths of increasing and decreasing subsequences at each position.

For each element, we update these variables by comparing with the previous element:

  • If current element > previous element: inc = dec + 1
  • If current element < previous element: dec = inc + 1
  • If current element = previous element: no change in inc or dec
