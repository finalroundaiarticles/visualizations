Dynamic Programming: Longest Increasing Subsequence

This visualization shows how Dynamic Programming is used to find the Longest Increasing Subsequence (LIS) in an array. We maintain a DP dictionary where each key is an array element and the value represents the length of the longest increasing subsequence ending at that element.

As we process each element, we look at all previously processed elements. If the current element is greater than a previous element, we can potentially extend that subsequence.
