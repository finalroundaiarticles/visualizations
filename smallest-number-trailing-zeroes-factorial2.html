Binary Search Visualization

Binary Search is a search algorithm that finds the position of a target value within a sorted array.

In this example, we're looking for the value n = 5 where n is the number of trailing zeros in n!.

Starting search space: [0, 25]

We'll check values to find where n = 5 (value with 5 trailing zeros)

First step: We check the middle element mid = 12.

12! has 2 trailing zeros, which is less than our target of 5.

Since we need more zeros, we search in the right half.

New search space: [13, 25]

12! has 2 trailing zeros (less than 5, so we move right)

Next, we check the middle element mid = 19.

19! has 3 trailing zeros, which is still less than our target of 5.

We need more zeros, so we continue searching in the right half.

New search space: [20, 25]

19! has 3 trailing zeros (less than 5, so we move right)

Now, we check the middle element mid = 22.

22! has 4 trailing zeros, which is still less than our target of 5.

We need more zeros, so we continue searching in the right half.

New search space: [23, 25]

22! has 4 trailing zeros (less than 5, so we move right)

Next, we check the middle element mid = 24.

24! has 4 trailing zeros, which is still less than our target of 5.

We need more zeros, so we continue searching in the right half.

New search space: [25, 25]

24! has 4 trailing zeros (less than 5, so we move right)

Finally, we check mid = 25.

25! has 6 trailing zeros, which is greater than our target of 5.

But since there's no number between 24 and 25, and 24! has 4 zeros (less than 5), we've found that 25 is the smallest number whose factorial has at least 5 trailing zeros.

Result: 25

25! has 6 trailing zeros (found our result!)
