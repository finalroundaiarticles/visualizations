Introduction to Catalan Numbers

Catalan numbers form a sequence of positive integers that appear in various counting problems. The first few Catalan numbers are:

C(0) = 1, C(1) = 1, C(2) = 2, C(3) = 5, C(4) = 14, ...

These numbers have fascinating applications in combinatorial mathematics and can be visualized in multiple ways.

Catalan Number C(1) = 1

For n=1, the Catalan number is 1, which represents:

Valid Parentheses:

There is only 1 way to arrange 1 pair of parentheses: ()

Binary Trees:

There is only 1 possible binary tree with 1 node.

Polygon Triangulation:

For a polygon with n+2 = 3 sides (triangle), there is 1 way to triangulate it (it's already a triangle).

Catalan Number C(2) = 2

For n=2, the Catalan number is 2, which represents:

Valid Parentheses:

There are 2 ways to arrange 2 pairs of parentheses: (()) and ()()

Binary Trees:

There are 2 possible binary trees with 2 nodes.

Polygon Triangulation:

For a polygon with n+2 = 4 sides (square), there are 2 ways to triangulate it.

Catalan Number C(3) = 5

For n=3, the Catalan number is 5, which represents:

Valid Parentheses:

There are 5 ways to arrange 3 pairs of parentheses: ((())), (()()), (())(), ()(()), ()()()

Binary Trees:

There are 5 possible binary trees with 3 nodes.

Polygon Triangulation:

For a polygon with n+2 = 5 sides (pentagon), there are 5 ways to triangulate it.

Catalan Number C(4) = 14

For n=4, the Catalan number is 14. We'll show just a few examples for each application:

Valid Parentheses:

There are 14 ways to arrange 4 pairs of parentheses, including: ((()())), ((())()), ((()))(), etc.

Binary Trees:

There are 14 possible binary trees with 4 nodes.

Polygon Triangulation:

For a polygon with n+2 = 6 sides (hexagon), there are 14 ways to triangulate it.
