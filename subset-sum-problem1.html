Binary Decision Tree for Subset Sum

We start with our problem: find a subset of numbers from [3, 4, 5] that adds up to 9.
For the first element (3), we have two choices: include it or exclude it. If we include 3, our new target becomes 9-3=6.
For the second element (4), we again have two choices from each previous state. Note that if we excluded 3 and included 4, our target becomes 9-4=5. If we included 3 and included 4, our target becomes 6-4=2.
For the third element (5), we again have two choices from each previous state. Only one path leads to a solution - exclude 3, include 4, include 5, which gives us 4+5=9. This path is highlighted in orange. When the target becomes 0, we know we've found a valid subset.
Solution Found! The highlighted path shows our successful subset: {4, 5} which sums to 9. This binary decision tree method explored all possible combinations systematically. The key was making binary decisions (include/exclude) for each element of the array.
